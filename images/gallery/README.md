# 율무 갤러리 이미지 (로컬)

이 폴더의 이미지는 Supabase 없이 **프로젝트 파일**로 제공됩니다.

## 폴더 구조

- `2023/` - 2023년 사진
- `2024/` - 2024년 사진  
- `2025/` - 2025년 사진

## 사용 방법

1. 각 연도 폴더에 이미지 파일(jpg, png, gif, webp)을 넣습니다.
2. 로컬 서버(`node server.js`)를 실행한 뒤 메인 페이지의 "갤러리" 섹션과 `/gallery.html`에서 확인할 수 있습니다.

기존 Supabase에 있던 이미지를 옮기려면, Supabase Storage에서 다운로드한 뒤 위 연도별 폴더에 넣으면 됩니다.
